# 4-Week Rolling Content Calendar

> A repeatable content system. Each week follows the same structure so you always know what to post, when to post it, and how to batch it.

---

## Weekly Content Structure

| Day | Content Type | Platform Priority | Hashtag Set | Posting Time |
|---|---|---|---|---|
| Monday | Education | IG Carousel + TikTok | Niche A + Education | 5-7 PM |
| Tuesday | Transformation | IG Reel + TikTok | Niche B + Transformation | 6-8 PM |
| Wednesday | Behind the Scenes | IG Stories + TikTok | Community + Niche A | 5-7 PM |
| Thursday | Testimonial / Social Proof | IG Feed + Facebook | Niche A + Testimonial | 5-7 PM |
| Friday | Personality / Fun / Trend | TikTok + IG Reel | Community + Trending | 6-9 PM |
| Saturday | Promo / Offer / Model Call | IG Feed + Stories + FB | Niche B + Promo | 10 AM-12 PM |
| Sunday | Batch + Plan (no posting) | — | — | — |

**Daily stories (every day, all week):** 2-3 stories minimum — mix of behind-the-scenes, polls/questions, client results, personal life.

## Content Batching Schedule (Sundays)

### Morning (1-2 hours): Plan
- Review last week's analytics (reach, saves, shares, profile visits, DMs)
- Identify top-performing content — plan to create similar content
- Write all captions for the week
- Select hashtag sets for each post
- Choose trending sounds for Reels/TikToks

### Afternoon (2-3 hours): Film
- Change outfits 2-3 times (so content looks like different days)
- Film in 2-3 different locations or backdrops
- **Must-film list:**
  - 2 talking-head clips (education + FAQ)
  - 1 before/after sequence (use a past client with permission or a model)
  - 2-3 trending Reel/TikTok concepts
  - 10-15 B-roll clips (hands working, products, workspace, tools)
  - 5-7 story clips (casual, personality, day-in-the-life feel)

### Evening (1 hour): Schedule
- Edit videos (CapCut or InShot for quick edits)
- Schedule all posts in Metricool for the week
- Queue up story content
- Set reminders for TikTok manual posts

## Model Call Schedule

- **Weeks 2 and 4**: Post model call (alternating services if you offer multiple)
- **Goal**: 2-3 model clients per model call
- **Content captured**: Use the before/after and testimonial content from model calls to fuel the following week's transformation and social proof posts

## Monthly Calendar Review Checklist

At the end of each 4-week cycle:

- [ ] What were my top 3 posts by reach?
- [ ] What were my top 3 posts by saves/shares?
- [ ] Which content type drove the most DMs/bookings?
- [ ] How many new followers this month?
- [ ] How many DM conversations turned into bookings?
- [ ] What content should I create more of?
- [ ] What content should I stop creating?
- [ ] Any new trending formats or sounds to incorporate?
- [ ] Update hashtag sets (drop underperformers, add new ones)
- [ ] Plan next month's seasonal content and promos
