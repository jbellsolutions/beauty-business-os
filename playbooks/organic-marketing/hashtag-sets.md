# Hashtag Sets Library

> Pre-built hashtag sets for beauty professionals. Copy-paste the set that matches your content type, then customize with your city and niche specifics.

---

## How to Use Hashtags

### Instagram Strategy
- Use **20-30 hashtags** per post
- Place them at the end of your caption (not in comments — algorithm treats them the same and captions are easier to manage)
- Mix sizes: 5 large (1M+), 10 medium (100K-1M), 10 small (10K-100K), 5 micro (under 10K)
- Rotate sets — do not use the exact same 30 hashtags on every post

### TikTok Strategy
- Use **3-5 hashtags** maximum
- Focus on specific, trending, and niche tags
- Always check the Discover page for trending hashtags in your space
- #fyp and #foryou are debated — use if you want but do not rely on them

### Facebook Strategy
- Use **3-5 hashtags** maximum (Facebook penalizes hashtag stuffing)
- Focus on branded and local tags only

## Hashtag Rotation Schedule

Do not use the same set every day. Rotate across 3-4 sets per week:

| Day | Content Type | Hashtag Set |
|---|---|---|
| Monday | Education | Niche Set A + Education tags + Location tags |
| Tuesday | Transformation | Niche Set B + Transformation tags + Location tags |
| Wednesday | Behind the Scenes | Community tags + Niche Set A + Location tags |
| Thursday | Testimonial | Niche Set A + Testimonial tags + Location tags |
| Friday | Personality | Community tags + Trending tags + Location tags |
| Saturday | Promo | Niche Set B + Promo tags + Location tags |

## Tracking What Works

- Check Instagram Insights > Content > See All > Sort by Reach to see which hashtag sets drove the most discovery
- If a post gets 30%+ reach from hashtags, that set is working — keep using it
- If a post gets under 10% from hashtags, swap out the large tags for more medium/small ones
- Refresh your hashtag sets monthly as trends change
