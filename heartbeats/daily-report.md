# Daily Report Heartbeat

**Trigger:** 6 PM local time, or when user says "how did I do today" / "daily report" / "end of day"

## Data Sources
1. **GHL:** Services completed, revenue, new leads, messages
2. **Content:** Posts published today, engagement if available
3. **Tasks:** What was completed vs planned

## Output Format
Concise wrap-up. Positive tone — end the day on a good note. Max 10 lines.

1. **Revenue** — Today's total with service count
2. **Highlights** — Best thing that happened (new client, good review, post that performed)
3. **Content** — What was posted, basic engagement if available
4. **Tomorrow's top 3** — What to focus on tomorrow (prep tonight if needed)

## Example Output
```
Great day, Glow Studio! 🌟

💰 TODAY: $430 — 3 services
- Volume Full Set ($250) — new client Sarah ⭐
- Lash Fill ($85)
- Lash Lift & Tint ($95)

🎉 HIGHLIGHT: Sarah loved her lashes — she already rebooked for a fill in 2 weeks!

📱 CONTENT: Posted the lash lift timelapse at 6 PM — 47 views in the first hour, 3 saves

📋 TOMORROW'S TOP 3
1. Send Sarah a review request (she was so happy — strike while it's hot!)
2. Film a quick "what I use on sensitive eyes" video (you mentioned this today)
3. 2 appointments tomorrow — prep by reviewing client notes tonight

Rest up — you earned it! 💜
```
