# Before/After Image Template

## Overview
Generates the FRAME and TEMPLATE for before/after comparison posts — not the actual client photos. The AI creates a branded split-screen layout with placeholder areas where the user drops in their own client photos. Before/after content is the highest-performing organic content type for beauty businesses.

## Dimensions
- **Instagram Feed:** 1080x1080 (split left/right)
- **Instagram Carousel:** 1080x1350 (swipe for the "after")
- **Story format:** 1080x1920 (top = before, bottom = after)

## Layout Spec

### Split Layout (Feed - 1080x1080)
- Left half: "Before" photo placeholder
- Right half: "After" photo placeholder
- Center divider: thin line or gradient fade in brand colors
- Top or bottom strip: brand name / logo area
- "Before" and "After" labels positioned clearly
- Clean border/frame around each photo area

### Carousel Layout (1080x1350)
- Slide 1: "Before" with full frame and label
- Slide 2: "After" with full frame and label
- Consistent branding across both slides
- Swipe indicator on slide 1

### Story Layout (1080x1920)
- Top half: "Before" with label
- Bottom half: "After" with label
- Horizontal divider with brand accent
- CTA at very bottom: "Book your transformation"

## Important Notes
- **The AI generates the FRAME only** — it does not generate fake before/after photos
- **Client photos are real** — user drops them in via Canva, Photoshop, or phone editor
- Always get client permission before posting before/after photos
- HIPAA considerations for med spa treatments — get written consent

## Post-Generation Workflow
1. AI generates the branded frame/template
2. User downloads the template
3. User opens in Canva
4. User drops in their real "before" photo on the left
5. User drops in their real "after" photo on the right
6. User adjusts text labels if needed
7. User adds caption with service details, pricing, booking link

## Canva Alternative
Since most beauty professionals already use Canva, offer both options:
- **Option A:** AI-generated custom frame (unique to their brand)
- **Option B:** Direct Canva template recommendations
  - Search "before after beauty" in Canva templates
  - Customize with brand colors and fonts
  - Save as reusable template

## Niche Variations
- **Lash extensions:** Close-up eye shots, dramatic difference
- **Hair color:** Full head shots, lighting consistency matters
- **Facials/skin:** Close-up skin texture, same lighting required
- **Nails:** Hand positioning, consistent background
- **Brows:** Close-up brow area, symmetry focus
- **Body contouring (med spa):** Consistent angles and distance
