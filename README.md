# Beauty Business OS

Your AI-powered business operating system. Built for beauty professionals.

## What This Is

A Claude Code project that runs your entire beauty business through conversation. You talk to it. It handles operations, content, marketing, appointments, client follow-up, and reporting.

No technical knowledge required. No software to learn. Just talk to your business.

## Install (3 Steps)

### Step 1: Install Claude Code

If you don't have Claude Code yet:
- **Mac:** Download from [claude.ai/download](https://claude.ai/download) or run `brew install claude-code`
- **Windows:** Download from [claude.ai/download](https://claude.ai/download)

### Step 2: Open This Project

Open your terminal (or Claude Code desktop app) and navigate to this folder:

```
cd beauty-business-os
claude
```

Or open the folder in VS Code/Cursor with the Claude Code extension installed.

### Step 3: Say "Set Me Up"

That's it. Claude will ask you 10-15 questions about your business, configure everything, and you're live.

## What You Can Say

Once set up, just talk to it:

| What You Want | What You Say |
|--------------|-------------|
| See your day | "What's on my schedule today?" |
| Create content | "Create this week's content" |
| Make a graphic | "Make me a promo for my lash special" |
| Check revenue | "How much did I make this week?" |
| Follow up leads | "Follow up with yesterday's leads" |
| Schedule posts | "Post this to Instagram at 6pm" |
| Get ideas | "What should I post today?" |
| Book someone | "Book Sarah for Thursday at 2pm" |
| Send reminders | "Remind tomorrow's clients about their appointments" |
| Reorder supplies | "Do I need to order anything?" |

## What's Inside

- **AI Business Manager** — Morning briefings, revenue tracking, task management, supply alerts
- **Content Creator** — Social posts, emails, blog posts, video scripts — all in your brand voice
- **Image Generator** — Promo graphics, before/after templates, story graphics
- **GHL Assistant** — Manages your GoHighLevel (leads, bookings, campaigns, follow-ups)
- **Auto-Posting** — Schedules and posts your content via Metricool
- **Dashboard** — Open `dashboard/index.html` in your browser for a visual overview

## Optional Integrations

These make the system more powerful but aren't required:

| Integration | What It Does | How to Connect |
|------------|-------------|----------------|
| GoHighLevel | CRM, bookings, campaigns | Paste API key during setup |
| Metricool | Auto-post to social media | Free signup → paste API key |
| Replicate | AI image generation (~$0.01/image) | Free signup → paste API token |
| Google Calendar | Appointment sync | Connect during setup |
| ManyChat | Instagram DM automation | Free signup → connect during setup |
