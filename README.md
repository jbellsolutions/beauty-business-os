# Beauty Business OS

Your AI-powered business operating system. Built for beauty professionals. Powered by Claude Code.

## What This Is

A turnkey product that runs your entire beauty business through conversation. Drop it into Claude Code, say "set me up," and you're live. It handles operations, content, marketing, appointments, client follow-up, CRM automation, and reporting — all by just talking to it.

No technical knowledge required. No software to learn. No code to write.

---

## Install (3 Steps)

### Step 1: Get Claude Code

If you don't have Claude Code yet:
- **Mac:** Download from [claude.ai/download](https://claude.ai/download) or run `brew install claude-code`
- **Windows:** Download from [claude.ai/download](https://claude.ai/download)
- **VS Code / Cursor:** Install the Claude Code extension

### Step 2: Open This Project

Open your terminal (or Claude Code desktop app) and navigate to this folder:

```
cd beauty-business-os
claude
```

Or open the folder in VS Code/Cursor with the Claude Code extension installed.

### Step 3: Say "Set Me Up"

That's it. The AI will:
1. Ask you 10-15 questions about your business (conversationally, not all at once)
2. Configure your CRM automatically (if you have GoHighLevel)
3. Open your dashboard in the browser
4. Offer to customize your dashboard with your brand colors
5. Give you 3 quick wins to start with

**You're live. Just talk to your business from now on.**
